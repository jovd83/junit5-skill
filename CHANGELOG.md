# Changelog

All notable changes to this repository will be documented in this file.

The format is based on Keep a Changelog and this project follows semantic versioning at the repository level.

## [1.0.0] - 2026-03-20

### Added

- Initial JUnit 5 skill family root entrypoint in `SKILL.md`
- Agent metadata in `agents/openai.yaml`
- Core implementation subskill for creating, running, evaluating, and correcting JUnit 5 tests
- Analysis subskill for deriving scenarios from requirements, code, and TDD, BDD, and plain-text definitions
- Architecture subskill for suite structure, fixture ownership, and naming
- CI subskill for Maven, Gradle, Console Launcher, and environment-specific execution guidance
- Migration subskill for JUnit 3 and JUnit 4 to JUnit 5 modernization
- Extensions subskill for callbacks, resolvers, and custom annotations
- Documentation subskills for test explanation and root-cause analysis
- Stakeholder reporting subskill
- IDE installer subskills for IntelliJ + Junie and VS Code + Codex
- Shared references, starter templates, release manifest, validator, and helper scripts
- MIT license and GitHub-facing README
