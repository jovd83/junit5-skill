# JUnit 5 Skill Family v1.0.0

## Summary

Initial public release of the JUnit 5 skill family.

## Included In This Release

- Root routing skill for general JUnit 5 requests
- Focused subskills for core authoring, analysis, architecture, CI, migration, and extensions
- Documentation and stakeholder-reporting helpers
- IDE setup guides for IntelliJ + Junie and VS Code + Codex
- Shared references, templates, validation, and release-manifest generation

## Primary Use Cases

- Create and correct JUnit 5 component, slice, repository, integration, and service tests
- Derive executable tests from requirements, source code, and narrative test-case definitions
- Modernize JUnit 4 suites incrementally
- Diagnose flaky or failing runs
- Configure and explain execution in local development and CI

## Notes

- The package is intentionally scoped to JUnit 5 and JUnit Jupiter compatible workflows.
- New guidance defaults to JUnit 5 patterns even if newer JUnit documentation sites are used as reference inputs.
